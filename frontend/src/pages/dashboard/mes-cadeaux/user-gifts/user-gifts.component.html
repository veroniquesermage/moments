🎁 Mes cadeaux

Les cadeaux sont triés par ordre de priorité. Utilisez la poignée ↕️ pour organiser votre liste.

Cliquez sur un cadeau pour consulter son détail, le modifier ou le supprimer.

Chaque ligne commence par des boutons pour modifier la priorité : utilisez Tab pour les afficher, puis les activer avec Entrée ou Espace.
